# Node Image Workbench

A personal web project built with Next.js, React, TypeScript, and Tailwind CSS.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run typecheck
```

## Project Structure

- `app/` - Next.js app routes and global styles
- `components/` - React UI components
- `public/` - Static assets
- `scripts/` - Local runtime patches and helper scripts

## Notes

Generated build output, dependency folders, local caches, logs, and downloaded package archives are ignored by Git.
